# DataAnalysis

## Project Overview

This repository contains a research-oriented data project that studies student dropout in higher education programs. It brings together exploratory analysis, visual storytelling, and geographic context to understand patterns of academic withdrawal with care and attention.

## What is included

- `notebooks/análise.ipynb`: exploratory data analysis and investigation of factors associated with dropout.
- `notebooks/gráficos.ipynb`: visualizations that illustrate emerging trends and support interpretation of the results.
- `src/coordenadas.py`: a small utility to convert Brazilian postal codes (CEP) into latitude/longitude coordinates using the Nominatim geocoding service.
- `outputs/figures/` and `outputs/maps/`: generated charts and interactive maps that document the analytic journey.

## Research focus

The project is designed as a thoughtful inquiry into academic attrition, using data-driven methods to reveal the dimensions of student disengagement and the spatial patterns that may accompany it. It is not only a technical exercise, but also a reflective process that values the knowledge gained through analysis.

## Why it matters

By combining quantitative analysis with geographic insight, this work aims to make the phenomena of higher education dropout more visible and better understood. The project showcases a careful approach to research and highlights the learning that comes from translating data into meaningful evidence.
